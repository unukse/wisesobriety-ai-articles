Welcome to the Auth Pages

This is the default landing page for your authentication flows.

How to use these pages:

  • Reset Password: This page should be accessed via the password reset link sent to your email. It expects access tokens in the URL.
  • Confirm Email: This page should be accessed via the email confirmation link sent to your email.

⚠️ Important Note:

If you're seeing "Missing access token" errors, it means you're trying to access the reset password page directly without the proper tokens. The reset password page should only be accessed via the email link sent by Supabase.
